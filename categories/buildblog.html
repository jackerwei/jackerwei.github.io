搭建博客和html的预览

如何用 hugo 搭建个人博客

1.首先我们先下载hugo,我系统是win10，下载64位windows版本 hugo releases然后把下载的文件进行解压，解压后把hugo.exe的路径添加到path路径中 。

2.然后我们通过cmder来测试hugo是否安装成功，cmder下载 cmder download

3.打开cmder bash的终端，然后运行下面命令： hugo version

4.创建一个新的站点 hugo new site jackerwei.highub.io-creator

5.添加主题 cd jackerwei.highub.io-creator git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml

6.添加一些内容 hugo new posts/my-first-post.md 可以编辑文件内容 --- title: "My First Post" date: 2019-03-26T08:47:11+01:00 draft: true ---

7.启动hugo服务器 hugo server 在http：// localhost：1313 /导航到新站点。

8.站点配置 config.toml在文本编辑器中打开： baseURL = "https://example.org/" languageCode = "en-us" title = "My New Hugo Site" theme = "ananke" 通过修改theme来设置主题，本地不需要对baseURL进行配置，如果是对面提供服务，则需要配置 域名

9.构建静态页面 在当前目录下创建一个public文件夹，然后通过 hugo 来创建静态文件
